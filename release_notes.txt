## luci-app-picoclaw v1.0.0

First release of the LuCI management interface for PicoClaw AI assistant on OpenWrt routers.

### What's included
- LuCI Controller + View with 5-language support (EN/CN/JA/DE/PT)
- IPK package for **OpenWrt 24.10** (opkg format)
- APK build script for **OpenWrt 25.xx** (apk-tools format)
- One-click Python installer (works for all versions)
- init.d procd service script

### Install

**OpenWrt 24.10** (opkg / .ipk):
```
opkg install luci-app-picoclaw_24.10_all.ipk
```

**OpenWrt 25.xx** (apk / .apk):
```
chmod +x build-apk-25xx.sh
./build-apk-25xx.sh
apk add --allow-untrusted /root/luci-app-picoclaw_1.0.0_all.apk
```

**Any version** (recommended):
```
python install_picoclaw_luci.py
```

### Requirements
- PicoClaw must be installed first: https://github.com/sipeed/picoclaw
- OpenWrt 24.10 or 25.xx with LuCI
